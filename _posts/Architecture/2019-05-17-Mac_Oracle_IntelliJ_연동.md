# Environment
- MacOS

# Oracle IntelliJ 연동
- <oracle_home>/jdbc/ 폴더에 ojdbc6 복사
- <java_home>/jdk1.8.0_201/jre/lib/ext 외부전용 폴더에 ojdbc6 저장
- 혹시 모르니 <java_home>/jre1.8.0_201/jre/lib/ext 폴더에도 ojdbc6 저장
