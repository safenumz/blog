리스트(List)¶

  • 자료형이 다른 여러 벡터들로 구성된 자료 구조
  • 벡터는 동일한 자료형만 가능하나(자료의 동질성) 리스트는 다른 자료형으로 구성된 벡터들을 포함함(자료의 이질성)
  • 리스트는 또 다른 리스트를 포함할 수 있음 -> Recursive Vector

for (변수명 in 벡터) {

x <- 0
sum <- 0

    }
