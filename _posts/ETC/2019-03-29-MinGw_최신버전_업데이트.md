# MinGw_최신버전_업데이트

```shell
$ mingw-get upgrade mingw-get-bin

$ mingw-get upgrade mingw-get-lic

$ mingw-get upgrade pkginfo-bin

$ mingw-get upgrade gcc-core-lic

$ mingw-get upgrade gcc-core-bin

$ mingw-get upgrade gcc-g++-bin

$ mingw-get upgrade msys-base-bin

```

```shell
# 모든 패키지 한번에 업데이트
$ mingw-get update

$ mingw-get upgrade
```
