[WEB] JSP Object

Jun 17, 2019 • 3 min read

WEB

JSP가 제공하는 기본객체

  • request : 클라이언트의 요청 정보를 저장한다.
  • response : 응답 정보를 저장한다.
  • pageContext : JSP 페이지에 대한 정보를 저장한다.
  • session : HTTP 세션 정보를 저장한다.

  • 서버로 보낼 때
