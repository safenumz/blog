미디어쿼리

  • 특정 조건을 만족할 때만 화면 설정 반영
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
    <style>
      div {
        border:10px solid green;
        font-size:60px;
      }
      @media(min-width:800px) {
        div{
          display:none;
        }  
      }
    </style>
  </head>
  <body>
    <div>
      Responsive
    </div>
  </body>
</html>

화면의 크기가 800px 이상이면 display:none으로 변경

<style>
  @media(min-width:800px) {
    div{
      display:none;
    }
  }
</style>

CSS 코드의 재사용

  • style.css 파일을 불러와서 스타일을 씀
  • style.css 파일만 수정하면 전체 웹페이지에 효과를 줄 수 있음
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <link rel="stylesheet" href="style.css">
  </head>
</html>
