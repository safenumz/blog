[WEB] 02) 표 만들기(table)

Aug 22, 2018 • 1 min read

WEB

Table

td, tr

  • td : table data
  • tr : table row
  • border : 숫자가 커질 수록 두꺼워 짐
  • 심미성을 신경쓴다면 css를 사용해야 함

thead, tbody, tfoot

  • th : thead 부문 문자 진하게

셀 병합

  • rowspan : 수직으로 병합
  • colspan : 수평으로 병합
<!DOCTYPE html>
<html>
<body>
<table border="2">
  <thead>
    <tr>
      <th>이름</th><th>성별</th><th>주소</th><th>회비</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>최진혁</td><td>남</td><td rowspan="2">청주</td><td>1000</td>
    </tr>
    <tr>
      <td>최유빈</td><td>여</td><td>500</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">합계</td><td>1500</td>
    </tr>
  </tfoot>
</table>
</body>
</html>
