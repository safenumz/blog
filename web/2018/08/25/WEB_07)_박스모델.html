박스모델

박스 그리기

<style>
  /*
  block level element
  */
  h1 {
    border-width:5px;
    border-color:red;
    border-style: solid;
    /* display:inline; */
  }
  /*
  inline element
  */
  a {
    border-width: 5px;
    border-color:red;
    border-style: solid;
    display:block;
  }
</style>

이렇게 간단하게 쓸 수도 있다

<style>
  h1, a {
    border: 5px solid red;
  }
</style>

간격 설정

  • padding : 테두리와 content 사이의 간격
  • border: 테두리 굵기
  • margin: 테두리와 테두리 사이의 간격
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
    <style>
      h1, a {
        border: 5px solid red;
        padding:20px;
        margin:20px;
        display:block;
        width:100px;
      }
    </style>
  </head>
  <body>
    <h1>CSS</h1>
    <h1>CSS</h1>
  </body>
</html>
